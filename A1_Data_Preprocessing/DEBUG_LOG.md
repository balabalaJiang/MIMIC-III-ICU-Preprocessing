以下紀錄在完成本次MIMIC-III前處理作業時，所遇到的技術問題與解決過程：

## 紀錄一：環境未安裝Seaborn套件
* 遇到問題的環節：區塊0環境設定與資料讀取 - 載入套件時。
* 錯誤訊息 / 異常現象：執行`import seaborn as sns`時，程式中斷並出現`ModuleNotFoundError: No module named 'seaborn'`錯誤。
* 原因分析：目前的Python虛擬環境中，一開始沒有安裝`seaborn`這個用於繪製統計圖表的套件。
* 解決方案：在終端機(Terminal)執行`!pip install seaborn`進行安裝。安裝完成後重新執行載入套件的程式碼，即成功匯入。我也順便將其加入到了`requirements.txt`中。
* 狀態：✅ 已解決

## 紀錄二：讀取檔案時字串格式錯誤導致找不到檔案
* 遇到問題的環節：區塊1.1讀取資料與查看欄位時。
* 錯誤訊息 / 異常現象：出現`FileNotFoundError: [Errno 2] No such file or directory`，表示找不到檔案。
* 原因分析：在設定路徑變數並嘗試讀取檔案時，字串前面忘記加上`f`來啟用格式化字串(f-string)，導致變數名稱沒有被正確解析轉換成實際的路徑，程式因此去尋找錯誤的字面名稱，找不到`mimic3c.csv`檔案。
* 解決方案：將讀取路徑的字串寫法修正，加上`f"..."`前綴，讓路徑變數能被正確讀取。修改後成功讀入資料並印出欄位。
* 狀態：✅ 已解決
